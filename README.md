# Supa Coffee — Франшиза кофейни

Профессиональный сайт франшизы кофейни **Supa Coffee**. Современный адаптивный дизайн с плавными анимациями и полной функциональностью. 

**Посмотреть сайт:** https://rasul-max-oss.github.io/Supa-Coffee.-Landing-page/

## Демо

Попробуйте сайт прямо сейчас: https://rasul-max-oss.github.io/Supa-Coffee.-Landing-page/

## Особенности

- **Полный адаптив** для всех размеров экранов (мобильные, планшеты, десктоп)
- **Плавные CSS анимации** для улучшения UX
- **Интерактивная форма** с валидацией и модальным окном благодарности
- **Семантический HTML5** код
- **BEM методология** в стилях
- **Быстрая загрузка** оптимизированных изображений
- **SEO оптимизация** с Meta Tags, Open Graph и Structured Data
- **Sitemap и robots.txt** для поисковых систем

   - Просто откройте `index.html` в вашем браузере
   - Или используйте Live Server в VS Code

## Развертывание на GitHub Pages

Сайт автоматически развернут на GitHub Pages по адресу:
https://rasul-max-oss.github.io/Supa-Coffee/

Для использования с вашим аккаунтом:

1. Сделайте fork репозитория
2. В настройках репозитория (Settings → Pages) выберите Branch: main
3. GitHub автоматически опубликует сайт по адресу:
   `https://your-username.github.io/Supa-Coffee/`

## CSS анимации

Используемые эффекты для плавных переходов:

```css
fadeInUp      /* Появление с прокруткой вверх */
fadeIn        /* Плавное появление */
slideInLeft   /* Скольжение слева */
slideInRight  /* Скольжение справа */
scaleIn       /* Масштабирование */
```

## BEM структура

Все классы следуют методологии BEM (Block, Element, Modifier):

```css
.benefits__card              /* Block + Element */
.benefits__card:hover        /* Modifier (state) */
.formats__card-name          /* Element of Element */
```

## Как добавить новый функционал

### Добавить новую секцию:

1. Создайте HTML в `index.html`
2. Добавьте стили в `styles.css` с BEM классами
3. Оберните в `<section class="section-name">` для автоматической анимации

### Добавить JavaScript функцию:

1. Добавьте код в `script.js`
2. Используйте современный ES6+ синтаксис
3. Добавьте комментарии для объяснения
